# vue-cli3-lerna-ui

## Project setup

``` bash
vue create --preset ziyi2/vue-cli3-lerna-ui my-project --packageManager yarn
```

### Compiles and hot-reloads for development

``` bash
yarn serve
```

### Build UI Library

``` bash
lerna run lib
```

### Publish UI Librayr

``` bash
lerna publish
```

### Compiles and minifies for production

``` bash
yarn build
```

### Lints and fixes files

``` bash
yarn lint
```
